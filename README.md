# Hierarchy table
DEMO: [https://ankerek.github.io/hierarchy-table/](https://ankerek.github.io/hierarchy-table/)
## Installation

```sh
npm install
```
or
```sh
yarn install
```

## Development

- `npm start` run webpack dev server

## Build

- `npm run build` build files to `static` folder
- open `static/index.html`
